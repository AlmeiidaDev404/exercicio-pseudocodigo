# Exercícios de Algoritmos

Este repositório contém exercícios de lógica de programação feitos em Visualg.

## Conteúdo

- Simulador de CDB
- Viagem para Copa
- Filtro de Candidato
- Categorização de Atletas
- Verificador de Dividendos
- Sistema de Cashback
- Validador de Pix
- Análise de Criptoativos
- Imposto de Importação
- Ranking de Streaming

## Objetivo

Praticar lógica de programação e estrutura de decisão.
